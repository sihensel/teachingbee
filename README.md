## Set source directory for Python

```json
"terminal.integrated.env.osx": {
    "PYTHONPATH": "${workspaceFolder}/backend",
},
"terminal.integrated.env.linux": {
    "PYTHONPATH": "${workspaceFolder}/backend",
},
"terminal.integrated.env.windows": {
    "PYTHONPATH": "${workspaceFolder}/backend",
},
"python.envFile": "${workspaceFolder}/.env"
```
